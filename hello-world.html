Go por Exemplo: Olá Mundo

Nosso primeiro programa irá imprimir a clássica mensagem “olá mundo”. Aqui está o código completo.

 
package main

import "fmt"

func main() {
    fmt.Println("olá mundo")
}

Para executar o programa, coloque o código em ola-mundo.go e use go run.

 
$ go run ola-mundo.go
olá mundo

Às vezes nós queremos construir nossos programas em binários. Nós podemos fazer isso usando go build.

 
$ go build ola-mundo.go
$ ls
ola-mundo	ola-mundo.go

Podemos, então, executar o binário construído diretamente.

 
$ ./ola-mundo
olá mundo

Agora que podemos executar e construir programas básicos Go, vamos aprender mais sobre a linguagem.
